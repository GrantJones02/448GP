Throughout the lesson on the types of pieces,
a board set up in the starting position was used.

To provide a formal instruction, however,
here are the basic steps to set up the board.

Setting up the Board

1. Each player has their own edge,
where the leftmost square should be dark
and the rightmost square should be light.

2. The pawns go on the row in front of the edge.

3. On each player's back row,
in order from the corners to the center is
rook, knight, bishop, and then king/queen.

4. Your queen starts on her own color.

Pawns

Pawns have the most limited movement of any piece.

A pawn can only move forward a single square normally,
but if it is still in the starting position,
it can move two squares forward in a single turn.

The pawn is also the only piece that cannot
capture in the same way that it moves.

The pawn captures forward diagonally.
